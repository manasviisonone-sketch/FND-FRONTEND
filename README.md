# Fake News Detection Frontend

This project is a frontend application for detecting fake news articles. It allows users to input news articles and receive analysis on their authenticity.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/fake-news-detection-frontend.git
   ```
2. Navigate to the project directory:
   ```
   cd fake-news-detection-frontend
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

1. Start the development server:
   ```
   npm start
   ```
2. Open your browser and go to `http://localhost:3000` to view the application.

3. Input a news article in the provided form and click the submit button to check its authenticity.
